FF\Forms\Form::__callStatic

Magic wrapper for creating new fields

Use the first part of the fields class name (stripping the 'Field' suffix) with a lower-cased first letter as magic method. Any arguments passed to the magic method will be passed to FieldsFactory::create() in the given order. The first argument should always be the field's name. Examples: <code> Form::text('foo') <=> new \FF\Forms\Fields\TextField('foo'); Form::radio('foo', ['a', 'b']) <=> new \FF\Forms\Fields\RadioField('foo', ['a', 'b']); </code>
