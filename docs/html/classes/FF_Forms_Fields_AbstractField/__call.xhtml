FF\Forms\Fields\AbstractField::__call

Magic wrapper for adding constraints to a field

Use the first part of the constraints class name (stripping the 'Constraint' suffix) with a lower-cased first letter as magic method. Any additional arguments passed to the magic method will be passed to the constraint's constructor in the given order. Examples: <code> $myField->required() <=> $myField->addConstraint(new \FF\Forms\Constraints\RequiredConstraint()) $myField->minLength(8) <=> $myField->addConstraint(new \FF\Forms\Constraints\MinLengthConstraint(8)) </code> Can be chained. Example: <code> $myField->required()->minLength(8); </code>
